Fallback Launcher

When a workspace doesn't exist, this URL will be used to create a new one. If Coder API is not configured, this will be the primary launch method.
Available placeholders:
  • {ssh_url} — SSH clone URL (e.g., git@github.internal.com:user/repo.git)
  • {branch} — Current branch or ref name
  • {repo} — Repository name only
  • {owner} — Repository owner/organization
